# SDK Installation
tags: sdk, node, python, windows, install

Official SDK packages:

- Node.js: `@devportal/sdk`
- Python: `devportal`

Minimum versions:

- Node.js 18 or newer.
- Python 3.10 or newer.

Windows troubleshooting:

- Use PowerShell or Windows Terminal.
- Run `node --version` or `python --version`.
- Clear package manager cache if installation fails.
- If a corporate proxy is used, configure `HTTPS_PROXY`.

When reporting SDK issues, include OS, runtime version, package manager version, SDK version, and the full install error.
